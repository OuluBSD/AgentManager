# Guess The Number Game

A simple console game where players try to guess a randomly generated number within a specified range.

## Game Features

- Configurable number range (default: 1-100)
- Limited number of attempts (default: 7)
- Feedback on whether the guess is too high or too low
- Play again option

## How to Play

1. Run the game using Node.js
2. Enter a number within the given range
3. Receive feedback if your guess is too high or too low
4. Try to guess the number within the allowed number of attempts

## Installation

1. Make sure you have Node.js installed
2. Clone this repository
3. Navigate to the project directory
4. Run the game:
   ```bash
   npx tsx src/guess-the-number-game.ts
   ```

## Development

To run in development mode:
```bash
tsx watch src/guess-the-number-game.ts
```
