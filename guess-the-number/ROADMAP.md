# Guess The Number Game

A simple console game where you try to guess a randomly generated number.

## Project Roadmap

### Phase 1: MVP
- Basic number guessing functionality
- Input validation
- High/low feedback
- Win/lose conditions

### Phase 2: Refine UX
- Better visual feedback
- Game statistics
- Improved input handling

### Phase 3: Add Scoring
- Points based on attempts used
- Leaderboard functionality
- Difficulty levels
